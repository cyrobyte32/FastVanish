# FastVanish 💨
**A Simple Lightweight Vanish Plugin for Minecraft Servers**

---

## 🌟 Overview
This Plugin Allows to fully vanish from other non-admin players.
## ✨ Key Features
* **Vanish** Vanishing allows admins to become completely invisible from normal players.
* **Technical Highlight:** Players are completely invisible, hidden from the tablist and can not be detected by normal players.

## 🛠 Technical Stack
This plugin is built with a focus on performance and maintainability.

| Component | Technology / Version |
| :--- | :--- |
| **Java Version** | 17 / 21 |
| **Server Software** | Paper / Spigot (1.21.x) |
| **Build Tool** | Maven |


## 📜 Commands & Permissions
| Command | Description | Permission |
| :--- | :--- | :--- |
| `/v` | Vanishes yourself | `vanish.vanish` |
| `/v <player>` | Vanishes a set Player that does not require Permission | `vanish.vanish` |

---

## 🚀 Development
1. Clone the repository.
2. Import as a Maven project.
3. Build using `mvn clean package`.
